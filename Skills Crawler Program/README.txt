Make sure you have:
1、 Mysql & DBeaver
2、 Python 10+
3、 Chrome Browser (Download the driver corresponding to the browser version: http://npm.taobao.org/mirrors/chromedriver/
      Place chromedriver. exe in the Google directory and Python installation directory)


Usage Steps:
1、Open install-dependencies.bat. Wait until all is done
2、Modify the code in crawler. py and connect to the MySQL database
3、Now, you can start crawling by opening start.bat
